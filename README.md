# MapAid Online Water Library ("OWL")

OWL is a web library for browsing and searching a classified archive of African
water-infrastructure documents — wells, boreholes, water points, and community
access records.

The aim is to turn scattered field reports, spreadsheets, and handwritten
assessments into accessible, searchable data, so that water infrastructure needs
across the continent can be seen, understood, and acted on. It is built as part of
a collaboration between Global MapAid and Databricks (see SUDAAK Project).
